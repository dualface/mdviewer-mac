# MDViewer Mac

Native macOS Markdown viewer built with SwiftUI and WKWebView. It is based on the rendering behavior of `../mdviewer`, but uses a desktop app shell instead of a local Go HTTP server.

## Features

- Open a Markdown file or a folder workspace.
- Browse the workspace with a native sidebar.
- Preview Markdown, images, and text/code files.
- Render Mermaid diagrams, KaTeX math, tables, task lists, footnotes, and definition lists.
- Syntax highlighting via highlight.js.
- Resolve relative images and Markdown links inside the opened workspace.
- Multi-tab preview, refresh, dark/light/system theme, font size, and preview width settings.
- Sandbox-friendly read-only access through user-selected files and folders.

## Development

Install renderer dependencies and build the offline WebView bundle:

```bash
cd Renderer
npm install
npm run build
cd ..
```

Generate the Xcode project:

```bash
xcodegen generate
```

Build and test:

```bash
xcodebuild -project MDViewerMac.xcodeproj -scheme MDViewerMac -destination 'platform=macOS' build
xcodebuild -project MDViewerMac.xcodeproj -scheme MDViewerMac -destination 'platform=macOS' test
```

Open a fixture workspace from the adjacent reference project:

```bash
open -a ~/Library/Developer/Xcode/DerivedData/MDViewerMac-*/Build/Products/Debug/MDViewerMac.app ../mdviewer/testdata
```

## Notes

`App/Resources/Renderer` is generated by the renderer build and is bundled into the app so Markdown rendering works offline. Rebuild it after changing files under `Renderer/src`.
